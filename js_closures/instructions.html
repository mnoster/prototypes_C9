DOM - variable tie in: custom attributes

  1. Use the array pets in pets.js to dynamically fill the div #row_display
  2. Apply a click handler to the click event of each .data_row as you make it.
  3. Also add a custom attribute to each .data_row: 'data-index'
  4. In the click handler, find the 'data-index' attribute of the element that was clicked. Use it to find the index of the pets array element associated with the row
  5. Once you get the array element, use it to fill in the respective elements of the #detail_display div

DOM - associated variable

  1. Once you get the previous section to work, now attempt to get the same index by accessing the for loop's variable to get the associated index.
  2. console.log the value of the index from the for loop when the click handler is clicked
  3. Note that the value, no matter which element you click on, is equal to the same number: the last value of the for loop that caused it to end.
  4. Take the section of the code where you create the click handler itself. Move that click handler creator into its own function. Remember to pass in the appropriate variables to create the click handler, including the dom element to apply the click handler to, as well as the for loop index at that moment of creation
  5. In the click handler, access the local version of the index.
  6. Run the code and test the click handler. The index should now be correct.
